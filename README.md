# RADIUS REDACTED ⭕

**RADIUS REDACTED** is a high-octane, top-down tactical battlegrounds game built entirely with HTML5 Canvas and JavaScript. Set in a minimalist, "classified" digital arena, players must outmaneuver AI operatives while the combat perimeter—the **Radius**—collapses around them.

## 🕹️ Live Demo
👉 [CLICK HERE TO PLAY](https://behrozkhan908-hash.github.io/RADIUS-REDACTED/)

## 🚀 Key Features
* **Dynamic Radius Logic**: A custom-coded shrinking circular zone.
* **Operative Skins**: Choose from Ghost, Viper, and Titan.
* **Advanced AI**: 29 automated operatives designed to hunt the player.
* **CRT/Glitch Aesthetics**: A unique visual style featuring scanlines and minimalist UI.

## 🎮 Controls
* **W / A / S / D**: Navigate Operative
* **Mouse Move**: Aim Signal
* **Left Click (LMB)**: Neutralize Targets
