# RADIUS REDACTED ⭕

**RADIUS REDACTED** is a high-octane, top-down tactical battlegrounds game.

## 🕹️ Live Demo
[CLICK HERE TO PLAY](https://behrozkhan908-hash.github.io/RADIUS-REDACTED/)

## 🚀 Key Features
* **Dynamic Radius Logic**: A custom-coded shrinking circular zone.
* **Operative Skins**: Choose from Ghost, Viper, and Titan.
* **Advanced AI**: 29 automated operatives designed to hunt you.

## 🎮 Controls
* **W / A / S / D**: Move
* **Mouse**: Aim
* **Left Click**: Shoot
