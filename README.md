RADIUS REDACTED ⭕
RADIUS REDACTED is a high-octane, top-down tactical battlegrounds game built entirely with HTML5 Canvas and JavaScript. Set in a minimalist, "classified" digital arena, players must outmaneuver AI operatives while the combat perimeter—the Radius—collapses around them.

📝 Game Lore
In a world where digital identities are strictly controlled, you are a "Redacted" operative. Your mission is simple: be the last signal remaining. As the system initiates a Symmetry Breach, the playable area shrinks, forcing survivors into a lethal final confrontation.
🚀 Key Features
• Dynamic Radius Logic: A custom-coded shrinking circular zone that forces constant movement and tactical positioning.
• Operative Skins: Choose from three distinct tactical profiles: Ghost, Viper, and Titan.
• Advanced AI: 29 automated operatives with pathfinding and engagement logic designed to hunt the player.
• CRT/Glitch Aesthetics: A unique visual style featuring scanlines, chromatic aberration effects, and a minimalist "classified" UI.
• Zero-Dependency Engine: Built from scratch using vanilla JavaScript and HTML5 Canvas (no external game engines required).
🎮 Controls
• W / A / S / D: Navigate Operative
• Mouse Move: Aim/Direct Signal
• Left Click (LMB): Neutralize Targets
• Objective: Be the last operative alive as the Radius collapses.
🛠️ Technical Details
• Language: JavaScript (ES6+)
• Rendering: HTML5 Canvas API
• Styling: Tailwind CSS & CSS3 Animations
• Resolution: Fully responsive (adapts to any screen size)
